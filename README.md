# Marvel Information Portal

Учебный проект — информационный портал по вселенной Marvel.  
Приложение позволяет просматривать персонажей и комиксы, открывать подробную информацию о героях, получать случайного персонажа и искать персонажей по имени.

## Функциональность

- просмотр списка персонажей Marvel;
- загрузка дополнительных персонажей по кнопке `Load more`;
- отображение подробной информации о выбранном персонаже;
- список комиксов Marvel;
- отдельная страница с информацией о персонаже;
- блок случайного персонажа;
- возможность загрузить другого случайного героя;
- поиск персонажа по имени;
- валидация формы поиска;
- отображение сообщения, если персонаж найден или отсутствует;
- состояние загрузки с использованием Skeleton;
- обработка ошибок при запросах к API;
- адаптивная верстка интерфейса.

## Страницы проекта

### Главная страница

На главной странице отображаются:

- случайный персонаж;
- список персонажей;
- информация о выбранном герое;
- список комиксов выбранного персонажа;
- форма поиска героя по имени.

### Страница комиксов

Содержит список комиксов с:

- обложкой;
- названием;
- стоимостью;
- кнопкой загрузки дополнительных элементов.

### Страница персонажа

Отображает:

- изображение персонажа;
- имя;
- описание;
- декоративный баннер Marvel.

## Состояния интерфейса

В проекте предусмотрены разные состояния:

- персонаж еще не выбран;
- загрузка данных;
- персонаж успешно найден;
- персонаж не найден;
- ошибка заполнения формы;
- ошибка получения данных от API.

## Используемые технологии

- React;
- JavaScript;
- React Router;
- Formik;
- Yup;
- SCSS;
- Marvel API;
- Fetch API или Axios.
